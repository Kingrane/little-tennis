# Little Tennis

Minimalist 3D table tennis game built with Three.js + React.

## Stack

- **Three.js** — 3D rendering
- **React** — UI & state
- **Vite** — dev server & build
- **Tailwind CSS** — styling
- Custom physics engine with Magnus spin drift

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
